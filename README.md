## 設計風格

- **主色**：舞光橘 (`#F58220`) + 深炭黑底
- **字型**：思源黑體 / 思源宋體 + Inter
- **互動**：粒子背景、滾動激活、卡片翻轉、即時答題、AI 對話模擬
