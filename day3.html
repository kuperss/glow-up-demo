TOPIC 01 · 客戶服務一條龍

客戶說「我要報價」
那一刻起，你就在跑流程

展晟業務最容易出包的不是「不會賣」，是「流程斷掉」 — 報價沒回、訂單沒進、出貨延遲、收款拖到。

講師問請每個人說：上一份工作有遇過「客戶氣噗噗、後來才發現是流程出包」的經驗嗎？
STEP 1-2 · 詢價 + 報價

報價不是丟一個數字，是給客戶一份決策依據

什麼要寫
完整報價單
客戶名稱／案件名／品項型號／數量／單價／總價／有效期（建議 14-30 天）／付款條件／交期。
毛利
底價要記住
主管會給你各品項的最低成交價（毛利下限）。報價低於底價必須走 特案申請單。
系統
ERP / 報價系統
用公司報價系統開單，不要在 LINE 上口頭報。口頭報出問題你說了算 — 系統留檔才是依據。
真實案例新人 A 用 LINE 報「特價 50 元」，客戶截圖後到 ERP 改 80 元被告 — 公司賠錢，業務記過。系統留檔不是麻煩你，是保護你。
STEP 3 · 訂單確認

客戶下單那一刻，你要三件事同時做

  1. 確認訂單細節 — 型號、數量、單價、交貨地址、收件人手機。一個欄位錯，整批貨就送錯。
  2. 檢查信用額度 — ERP 上看這客戶的應收餘額 + 票期。超額要先催收上一批，再進新單。
  3. 確認交期 — 跟業管確認庫存。沒貨不要硬接，提前告知客戶 = 信任，事後出包 = 失分。
✕ 新人常踩的雷
「客戶說要 100 套，那就先進系統，貨之後再講」 — 結果採購沒跟上，客戶等兩週還沒到貨。
✓ 老鳥的做法
下單前先 LINE 業管確認「庫存能不能出 100 套」，得到「OK」再進系統。10 秒鐘的事。
大秉現場示範打開 ERP 訂單畫面，逐欄解釋每個欄位的意義 + 漏填會發生什麼事。
STEP 4-6 · 備貨 → 出貨 → 驗收

出貨不是業務的事，但客戶氣的時候找的是業務

業務的責任
追進度 + 通知客戶
出貨前一天打電話／LINE 客戶：「明天上午到，請留人簽收」。沒通知 = 客戶撲空 = 客訴。
物流選擇
小單宅配 / 大單配送
3 件以下走 黑貓／新竹；棧板貨走公司車或大型物流。客戶在偏遠地區先確認運費誰出。
驗收
簽收 = 法律意義
客戶簽完字 = 視同貨物無瑕疵交付。教客戶當場開箱驗，有問題當場拒簽。
提醒客戶說「先放著我等下驗」 — 業務一定要要求客戶在送貨單上備註「未當場驗收，後續發現瑕疵保留追訴權」。否則之後扯不清。
STEP 7-8 · 開票 + 收款

「賣得出去」是初級業務，
「收得回來」才是合格業務

  • 票期管理 — 客戶開票通常 30 / 60 / 90 天。手機行事曆設提醒，到期前 3 天確認客戶有錢入帳。
  • 催收順序 — 過期 1 天：LINE 提醒；過期 7 天：電話；過期 14 天：直接拜訪 + 主管知會；過期 30 天：上報財會走法務流程。
  • 發票開立 — 公司發票 + 三聯式。客戶要二聯就要事先講，月底集中開。
  • 收款方式 — 客戶轉帳要主動傳對帳單；現金（罕見）要兩人見證 + 簽收條。
業務鐵則沒收到錢的訂單只是「借錢給客戶讓他用」 — 業績是「貨出 + 錢進」，缺一不可。

REPORTING · 三層報表

主管不會要你每天傳一堆 LINE，
但要看到三種報表

每日
拜訪紀錄
時間／客戶／談了什麼／下一步行動／預估金額。當天填，三天後叫你補你會忘。
每週
週報
本週業績達成 / 本週重點拜訪 / 下週計畫 / 卡關問題 — 主動講你卡哪，主管才能幫你。
每月
MOR 月報
月底 25 號交。包含當月業績、達成率、A/B/C 客戶分布、毛利結構、預估下月。是升等審查依據。
大秉提醒不要把報表當作主管監督你。報表是「你給未來的自己看的」 — 三個月後想知道為什麼這客戶丟單，只能靠當時的紀錄。

一份好的拜訪紀錄 =

✕ 沒用的紀錄
「11/5 拜訪 ABC 燈飾，老闆人不錯，會考慮我們的崁燈」

→ 三天後你看不懂這在寫什麼。
✓ 有用的紀錄

· 詢問索爾 25W 100 套是否有量大優惠
· 主推 Ra90 + 2 年保固差異化

· 預估：8 萬／毛利 25%
· 注意：林老闆下週要去日本，11/12 再追
公式時間／人／談什麼／下一步／預估／注意點 — 五個欄位填好就贏一半人。
TOPIC 03 · 表單作業 + 請假

三張表單，
學會了才能把錢花得對

展晟給業務的權限是 — 只要你能說清楚為什麼，公司願意給。
但你必須走對表單、簽對流程。

FORM 1 · 費用申請單

業務帶客戶吃飯、加油、買樣品 —
都靠這張單

用途
業務開銷報帳
客戶招待餐 / 通行費 / 樣品 / 印刷 / 雜支。不能報的：個人加油、私人聚餐。
流程
填表 → 主管核 → 財會核
事前申請（大額）/ 事後檢附發票（小額）。每月 25 號前交完，月底結算。
關鍵
發票 + 客戶名稱
發票統編 = 公司、抬頭 = 公司全名。招待要寫被招待客戶名 + 案件名，不寫財會打回。
勇哥案例新人請客戶吃 5,000 沒寫客戶名，財會退回三次。第四次寫「林老闆 / ABC 燈飾 / 索爾案 100 套」秒過。
FORM 2 · 異折單

客戶說「這批貨有問題，少算一點」
怎麼處理？

  1. 先確認問題 — 真的是品質瑕疵？運送損壞？還是客戶想凹？回辦公室拍照存證。
  2. 跟主管報告 — 異折金額多少？品項？原因？主管判斷是否核准。
  3. 填異折單 — 系統開異折單，主管簽核 → 財會處理。客戶帳上自動扣抵下次貨款。
  4. 絕對不要 — 自己答應客戶「我幫你少算 5 千」 — 沒走系統 = 自己賠錢 + 主管罵。
原則「給客戶台階下」是業務智慧，但所有金額調整必須走系統。沒走系統 = 公司不認 = 你自己吞。
FORM 3 · 特案申請單

當客戶條件超出標準 — 你需要這張單

什麼時候要
三種情境
① 報價低於底價（毛利不足）
② 量大要求超優惠（如 1,000 套以上）
③ 戰略性接單（虧本搶單，搶占案場）
必填
為什麼要批准
① 客戶背景（A 級／重點工程）
② 案件規模 + 後續潛力
③ 不批的話誰會搶走（競品威脅）
流程
主管 → 區經 → 業總
小額：主管核。中額：區經理。大額：業務總經理。事前申請，不能事後補單。
關鍵特案不是「我談不下來」的藉口，是「這個案值得公司投資」的提案。寫單時用提案的角度寫。
LEAVE · 請假作業

請假不是「打個 LINE 跟主管說」就好

事前流程
系統請假單
假別（事假／病假／特休／喪假）/ 起迄時間 / 代理人 / 待辦事項交接。三天前送出。
急事
當天請假
早上 9:00 前打電話跟主管講（不是 LINE）。回來後補請假單。不打電話 = 曠職。
代理人
交接才能放心請
A 級客戶聯絡方式 + 進行中的案 + 下單／出貨／收款待辦事項，全部給代理同事。回來主動感謝。
CH 00 已教過病假要病假證明（請參考 onboarding.html）。月底衝刺週原則上不安排休假，特殊狀況要找主管商量。
TOPIC 04 · BI 系統

數字會說話，
會看 BI 才能聽懂

Business Intelligence — 把全公司的銷售資料變成一張一張可以「看出規律」的圖表。
看不懂 BI 的業務 = 拿到雷達卻只會猜方向的駕駛員。

BI · 三件事

BI 給業務的三個答案

Q1
我這個月做得怎樣？
業績儀表板：當月達成率 / 與上月比 / 與去年同期比 / 在區域排名。一打開就看得到。
Q2
我的客戶誰最值錢？
客戶分析：A/B/C 級分布 / 月度貢獻 / 毛利率 / 拜訪頻率。把 70% 時間放在 A 級客戶上的依據。
Q3
什麼產品最好賣？
產品熱銷榜：本月銷量 TOP / 毛利貢獻 TOP / 退貨率高的品項。新品要不要推這個就靠它。
BI · 實際操作

這台電腦，現場示範一輪

  1. 登入入口 — 公司 BI 系統網址（大秉現場提供）+ 個人帳密。第一次登入要改密碼。
  2. 切到「我的儀表板」 — 看當月業績達成 / 應收帳款狀況 / 拜訪達成率。
  3. 看「客戶分析」 — 篩選你責任區，看 A 級客戶清單 + 最近一個月活躍度。
  4. 看「產品熱銷」 — 篩選本季 TOP 20 產品，記住排名變化。客戶問什麼最好賣，你才有答案。
  5. 匯出 Excel — 月底 MOR 報表，從 BI 匯出原始資料 + 自己加註解，比手動算快 10 倍。
大秉示範實際操作：登入→ 我的儀表板 → 客戶分析 → 產品熱銷 → 匯出。請新人各自登入跟著做一次。
BI · 真實案例

老鳥怎麼用 BI 找錢

✕ 不會用 BI
每個月底主管問「為什麼這月差 20%」
→ 你只能說「客戶都不下單」。

隔月還是不知道哪裡可以補。
✓ 會用 BI
月中看 BI 發現：
· A 級客戶 X 連續 2 月沒下單
· 上季熱銷的索爾，這月跌出 TOP 10
→ 主動拜訪 X / 推索爾促銷 / 月底逆轉。
給新人BI 不是月底交差用，是每週一上班第一件事打開看。你要比主管早發現問題。
TOPIC 05 · 跨部門協作

業務不是孤狼，
是團隊的前線

你拉到的每一張單，背後是業管／採購／財會／物流／工程一群人在跑。
會跟內部部門合作的業務，跑得遠得多。

業管 · OPERATIONS

業管 = 你的後方司令部

他們做什麼
訂單 + 客戶資料 + 出貨
把你 ERP 的訂單轉成實際的揀貨／包裝／物流安排。客戶資料、信用額度、票期管理也是他們在管。
什麼時候找他們
下單前 + 出貨追蹤
下單前：「庫存夠不夠？」／ 出貨後：「貨到哪？」／ 客戶問：「我的訂單編號？」
合作關鍵
給他們完整資料
客戶名 / 統編 / 收件人手機 / 地址 / 特殊指示（裝箱方式、要不要收件人簽）— 一次給全，他們省力你也省事。
大秉業管要每天下午跑出貨單，5 點後才丟急單會被白眼。真有急事先打電話打招呼，再丟單。
採購 · PURCHASING

採購控制公司命脈 — 庫存

他們做什麼
進貨 + 庫存 + 缺貨預警
跟工廠下訂、安排進貨、控管最低庫存量、缺貨時找替代品。每個品項的補貨週期都是他們在算。
什麼時候找
大單 + 缺貨
超過正常量的訂單（如 500 套以上）要先問採購，避免吃光庫存。客戶要急貨缺貨時也是找採購想辦法。
合作關鍵
提早通知 + 信用
案場確定下單就先 LINE 採購預告。「業務說這單一定會中」結果三次有兩次跳票，採購就不再相信你。
關鍵詞跟採購建立信用 = 承諾的單一定要中。一次跳票兩次扣分，第三次他們自己會把預留庫存收回去。
財會 · FINANCE

財會看每張單的真假

他們做什麼
應收 + 發票 + 傳票
追客戶錢有沒有進來、開立發票、處理你的費用核銷、月結結帳。業績達不達標他們都看得出來。
什麼時候找
客戶逾期 + 報帳問題
客戶過期沒付款 / 發票要重開 / 費用核銷被退 / 月結有疑問。每月 25 號是他們最忙時段。
合作關鍵
資料完整 + 守時
表單欄位填全（發票統編、客戶名、案件名）/ 25 號前交報帳。缺一個欄位 = 整單退回 = 自己重做。
勇哥財會不是「擋你錢的人」，是「確保公司不出包的人」。態度好一點他們會主動提醒你哪裡會被查稅。
CROSS-DEPT · 黃金法則

跟內部部門相處的三個原則

  1. 不把他們當服務生 — 業務不是天王。業管／採購／財會的時間跟你一樣寶貴。
  2. 事前溝通 > 事後補救 — 大單先告知、缺貨先預警、報帳先問清楚 — 90% 的內部摩擦都是「沒先講」造成的。
  3. 幫他們解決問題 — 客戶資料填完整、單據填正確、催款主動跟、結案主動回 — 你幫他們省力，他們幫你開綠燈。
大秉業務在公司被討厭通常不是因為「績效不好」，是因為「每次找人就是來丟事」。記住這三個原則，半年後你會發現整個公司都幫你跑。
TOPIC 06 · 公司車輛規範

公司車是工具，
不是給你開回老家的

展晟業務有公務車是大利，但出包率也最高。今天把規範講清楚，後面少踩雷。

CAR · 使用規範

用公務車的五件事

01 油資
公司油卡
配發油卡。加滿後保留收據黏在油資登記簿。月底交給業管核對里程數。
02 行車紀錄器
永遠開著
每天上車自動錄影。出事故的關鍵證物。檔案不能自己刪，需事故 SOP 處理。
03 保養
里程數提醒
5,000 km 換機油 / 10,000 km 大保養。儀表板會提示。提早一週跟業管預約保養廠。
04 罰單
