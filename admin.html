AI 設定

設定 LLM 提供者 + API Key · 學員端 AI 助教 / AI 對話演練都使用此設定
安全性說明（必讀）

本系統使用瀏覽器直接呼叫 AI API，API Key 會儲存於 Firestore，所有登入用戶（含學員）都讀得到。最壞情況：員工複製 key 自用。

緩解措施：(1) 在 API 提供商後台設使用上限與額度警示 (2) 定期更換 key (3) Production 環境建議升級為 Cloudflare Worker 代理（可隨時請工程師補上，key 將完全鎖在後端）

提供者與金鑰
告訴 AI「你是誰」、「應該怎麼回答」、「有哪些限制」。設好以後所有 AI 助教 / AI 對話演練都會套用這個基底人設。
測試對話
點上方「測試呼叫」會發送一則「你好，請介紹一下舞光 LED」測試訊息，看看設定是否正常。
