> 本文件為業務新人訓練 AI 助教的知識庫之一，涵蓋客戶分類、客戶分級、拜訪節奏、議價心理、五大異議處理、業務日常工作、報表系統、跨部門協作。

## 1. 六種客戶類型（業務面對的中間商）

舞光不直接賣終端消費者，業務面對的是「會幫舞光把貨往下推」的中間商。
**客戶結構不同，溝通邏輯就完全不同。**

### 1.1 燈飾店（B2C 取向，賣的是「美感」）

- **特性**：多為夫妻檔或家族經營，店內陳列吊燈、吸頂燈、美術燈
- **客人**：屋主、設計師、新婚換屋客
- **決策者**：老闆／老闆娘當場拍板
- **在乎**：好不好賣、毛利高不高
- **經營重點**：情感連結＞規格表，常拜訪刷臉

### 1.2 電料行（B2B 走量，賣的是「規格」）

- **特性**：水電師傅、工班的進貨點。崁燈、燈管、軌道、感應燈、緊急照明為主
- **走法**：走量、走規格、走價格
- **決策者**：老闆 + 櫃台
- **在乎**：好開、好裝、出貨快
- **主推**：索爾崁燈、燈管走量品

### 1.3 賣場通路（門市維護 · B&Q / 家樂福 / 特力屋）

- **特性**：總部採購談合約，業務做門市維護
- **工作內容**：補貨、排面、試燈、跟店長混熟
- **業績靠**：陳列＞折扣，端架＞堆頭
- **核心技能**：跟採購、店長三方關係

### 1.4 室內設計師（指定來源 · 不下單但會「指定」）

- **特性**：設計師指定舞光，案子就會經由燈飾店或工班進來
- **價值**：一個設計師可能帶來十個案場
- **在乎**：光的質感、Ra、保固穩
- **經營重點**：型錄、案例、體驗館導覽
- **主推**：拉斐爾、奧丁、馬爾

### 1.5 工程商 · 統包（大量低毛 · 建商 / 商辦 / 公共工程）

- **特性**：一次出量大、價格殺到底、帳期長（30-90 天）
- **建商案最忌**：貨到才發現規格不符
- **關鍵能力**：報價算精、催收貨款、規格 double check
- **客戶看重**：CNS / BSMI 認證、產品責任險
- **主推**：高天井、泛光、防潮

### 1.6 經銷商二代（數位轉型夥伴 · 新世代學院核心客群）

- **特性**：展晟自身就是從燈飾店起家，最懂二代接班的痛點
- **學院規模**：新世代學院 1,000+ 人次受訓的核心客群
- **個人特質**：懂數位、想轉型、有挫折感
- **經營重點**：邀請來體驗館、新品發表、學院共學
- **核心理念**：與經銷商共好、共榮、共學

## 2. A/B/C/D 客戶分級

**對所有客戶平等付出 = 對所有客戶都不夠。**
分級不是歧視，是把資源放到對的地方。把 70% 時間放在 A 跟潛力 B。

| 級別 | 定義 | 拜訪頻率 | 經營重點 |
|---|---|---|---|
| **A 級** | 年貢獻高 · 配合度好 · 主推舞光 | 每週 1 次 | 深度服務、新品優先試 |
| **B 級** | 年貢獻中等 · 有成長空間 | 每 2 週 1 次 | 推升至 A 級、培養忠誠度 |
| **C 級** | 偶爾下單 · 量小但有戰略意義 | 每月 1 次 | 維持關係、觀察轉型機會 |
| **D 級** | 三個月內無進展 · 考慮降級或放棄 | 季拜訪即可 | 釋出時間給更值得的客戶 |

### 重要提醒
**分級不是固定的。每季重新評估一次。**
C 升 B、B 升 A 是業務的真本事 — 讓更多客戶想跟你做生意，比保住舊 A 級還重要。

## 3. 拜訪節奏（進門 30 秒 → 離店 15 分鐘）

拜訪不是「有去就好」，是有目的、有觀察、有跟進。

### 3.1 拜訪前 · 三個準備
1. **看上次拜訪紀錄**（避免「上次跟你講過了」的尷尬）
2. **從 ERP 拉這家近期下單、逾期款、客訴**
3. **包包固定 5 樣**：名片、最新型錄、3 支主力新品實品、不良品退換流程表、計算機

### 3.2 拜訪中 · 進門 30 秒看四件事
1. **陳列**：舞光擺哪？有沒有點亮？端架被誰佔了？
2. **競品**：哪家又出新品？促銷貼紙寫多少？
3. **人流**：老闆忙不忙？忙就少廢話切重點
4. **氣氛**：老闆心情好不好？老闆娘在算帳就別硬聊

### 3.3 拜訪後 · 15 分鐘內，車上寫完
- 答應的事（樣品 / 報價 / 退換）寫進待辦，**有截止日**
- 下次拜訪要帶什麼立刻寫
- 當下回 LINE：「老闆剛剛講的 OO 報價，我明天中午前傳給您」

> 隔一天就忘一半 — 紀錄是給未來的自己用的，不是給主管看的。

## 4. 報價議價節奏（讓價有節奏，不讓也有方法）

**台灣客戶心理：沒殺到價就覺得吃虧。**
所以開價要有空間 · 讓價要分階段 · 底線要清楚。

### 4.1 三段讓價節奏

| 階段 | 動作 | 說明 |
|---|---|---|
| **第一刀** | 讓 3-5% | 展現誠意，但保留空間。不要第一刀就把底價報出去 |
| **第二刀** | 讓 1-2% | 已接近底線，幅度縮小。配合「我幫您去爭取」的話術 |
| **第三刀** | 換條件不再讓價 | 改送：陳列道具、樣品燈、行銷物料、教育訓練、帳期延長 |

### 4.2 經典議價情境話術

**客戶說：「飛利浦給我比你便宜！」**
→ 把對話拉回規格表：「老闆飛利浦哪一支型號？我們瓦數、Ra、保固比一比，光看單支價錢沒意義」。
**永遠在規格上比，不在情緒上比。**

**客戶說：「不給這價，下次不下了！」**
→ 「老闆我懂您壓力，這個價我真的扛不住。這次我幫您額外加 X 支樣品您去推推看，下次量上來我再幫您爭取」。
**換條件，不退底線。**

### 4.3 鐵則：底價背起來

**每個型號的成本、毛利空間、公司最低核可，業務必須記在腦子裡。**
- 在客戶面前打電話問主管底價 = 被看扁
- 真要請示，回車上算清再回報才是專業

## 5. 五大異議處理（客戶常說的五句話）

每一句客戶的反對，都有「客戶心理」、「應對重點」、「範例話術」三層拆解。

### EP.01 「你們比較貴」
- **客戶心理**：習慣性殺價、或真的有比過
- **應對重點**：不要降價，先問是跟誰比 → 把對話拉回規格、保固、服務的差異
- **話術**：「老闆是跟哪一家比？我們這支是 Ra ≧ 90 + 2 年保固 + 1500 萬產品責任險，你看看那家有沒有？」

### EP.02 「我已經賣慣 X 牌了」
- **客戶心理**：怕換品牌客人不認、怕教育員工太累、怕有後遺症
- **應對重點**：不要求換掉，只要求「多一個選擇」 → 跟現有 X 牌定位錯開，多一個利潤更高的故事可以跟客人講
- **話術**：「沒要您換，但我們這支可以當您的高階款，毛利高 5-8%，您可以推給比較講究的客人」

### EP.03 「你們保固我不信」
- **客戶心理**：以前被某品牌坑過、聽同行說過、實際遇過退換貨拖很久
- **應對重點**：拿出具體流程，不要空口說實在。**1,500 萬產品責任險 + 2 年保固是真實底氣**，給客戶看上個月的處理紀錄
- **話術**：「老闆，您看這個是上個月某某客戶換貨的紀錄，3 個工作天內換好。我們有 1500 萬產品責任險，您不用擔心」

### EP.04 「客人指定要 X 牌怎麼辦」
- **客戶心理**：不想為了你跟客人吵架
- **應對重點**：給老闆一套轉換話術，讓他自己用得順
- **話術**：「規格一樣、保固多一年、台灣品牌台灣保固」三句搞定

### EP.05 「LED 都差不多吧」
- **客戶心理**：覺得 LED 是大宗商品、沒差別、純比價
- **應對重點**：用一個現場可演示的差異打破他的認知 → **不要說、要演**
- **演示**：拿出 Ra80 vs Ra90 的色卡比較，看蘋果或人臉色，差異一目了然

## 6. 業務日常工作節奏

### 6.1 標準一日時間表

| 時段 | 內容 | 重點 |
|---|---|---|
| **08:30 · 晨會** | 同步進度 | 主管 5 分鐘宣布事項 / 業務報今日拜訪計畫 / 重點客戶問題討論 |
| **09:00-12:00 · 拜訪** | 外勤主戰場 | 3-5 家客戶。新客戶先 30 分鐘破冰，老客戶聊新案。**不要在咖啡廳發呆。** |
| **12:00-13:30 · 午餐** | 回公司 / 跟客戶吃 | A 級客戶就一週請吃一次，建立交情。預算上限走費用申請 |
| **13:30-17:00 · 拜訪 / 出貨** | 下午衝量 | 下午客戶比較有空談價、看樣品。也是出貨集中時段，要追物流回報 |
| **17:00-18:00 · 進辦公室** | 收尾 + 紀錄 | 建單、報表、寫拜訪紀錄、回 LINE 訊息 |
| **18:00 · 下班** | 該休息就休息 | 業務不是 24 小時待命。下班後客戶 LINE 急事處理，閒聊明天再回 |

> 業務人員上班時間 08:30-17:30。**週一／連假後首日：08:00 報到。**
> 通勤彈性 10 分鐘內。8:15 打卡 = 視為遲到 5 分鐘。

## 7. 報表系統（三層）

主管不會要你每天傳一堆 LINE，但要看到三種報表。

### 7.1 每日 · 拜訪紀錄
- 時間／客戶／談了什麼／**下一步行動**／預估金額
- 當天填，**三天後叫你補你會忘**

#### 拜訪紀錄五欄公式（一份好的拜訪紀錄）
✕ 沒用的：「11/5 拜訪 ABC 燈飾，老闆人不錯，會考慮我們的崁燈」（三天後你看不懂）

✓ 有用的：
```
11/5 14:30 拜訪 ABC 燈飾 · 林老闆
· 詢問索爾 25W 100 套是否有量大優惠
· 主推 Ra90 + 2 年保固差異化
· 林關心：對手 X 牌降到 380
· 下一步：11/8 前報價 + 樣品寄達
· 預估：8 萬／毛利 25%
· 注意：林老闆下週要去日本，11/12 再追
```

**公式：時間／人／談什麼／下一步／預估／注意點 — 五個欄位填好就贏一半人。**

### 7.2 每週 · 週報
- 本週業績達成 / 本週重點拜訪 / 下週計畫 / 卡關問題
- **主動講你卡哪，主管才能幫你**

### 7.3 每月 · MOR 月報
- 月底 25 號交
- 包含當月業績、達成率、A/B/C 客戶分布、毛利結構、預估下月
- **是升等審查依據**

> 不要把報表當作主管監督你。報表是「**你給未來的自己看的**」 — 三個月後想知道為什麼這客戶丟單，只能靠當時的紀錄。

## 8. BI 系統（三個答案）

Business Intelligence 系統 — 把全公司的銷售資料變成可看出規律的圖表。
**不會用 BI 的業務 = 拿到雷達卻只會猜方向的駕駛員。**

### Q1 · 我這個月做得怎樣？
業績儀表板：當月達成率 / 與上月比 / 與去年同期比 / 在區域排名。一打開就看得到。

### Q2 · 我的客戶誰最值錢？
客戶分析：A/B/C 級分布 / 月度貢獻 / 毛利率 / 拜訪頻率。
**把 70% 時間放在 A 級客戶上**的依據。

### Q3 · 什麼產品最好賣？
產品熱銷榜：本月銷量 TOP / 毛利貢獻 TOP / 退貨率高的品項。
新品要不要推這個就靠它。

### 操作要點
1. **登入入口** — 公司 BI 系統網址（請主管提供）+ 個人帳密
2. 切到「**我的儀表板**」 — 看當月業績達成 / 應收帳款狀況 / 拜訪達成率
3. 看「**客戶分析**」 — 篩選你責任區，看 A 級客戶清單 + 最近一個月活躍度
4. 看「**產品熱銷**」 — 篩選本季 TOP 20 產品，記住排名變化
5. **匯出 Excel** — 月底 MOR 報表，從 BI 匯出原始資料 + 自己加註解，比手動算快 10 倍

> BI 不是月底交差用，是**每週一上班第一件事打開看**。你要比主管早發現問題。

## 9. 客戶服務一條龍（8 步流程）

展晟業務最容易出包的不是「不會賣」，是「流程斷掉」 — 報價沒回、訂單沒進、出貨延遲、收款拖到。

```
詢價 → 報價 → 訂單 → 備貨 → 出貨 → 驗收 → 開票 → 收款
```

### Step 1-2 · 詢價 + 報價
**報價不是丟一個數字，是給客戶一份決策依據。**

- **完整報價單應有**：客戶名稱／案件名／品項型號／數量／單價／總價／**有效期（建議 14-30 天）**／付款條件／交期
- **毛利底線**：報價低於底價必須走**特案申請單**
- **系統留檔**：用公司報價系統開單，**不要在 LINE 上口頭報**

### Step 3 · 訂單確認（同時做三件事）
1. 確認訂單細節（型號、數量、單價、交貨地址、收件人手機）
2. 檢查信用額度（ERP 上看這客戶的應收餘額 + 票期）
3. 確認交期（**沒貨不要硬接**，提前告知客戶 = 信任）

### Step 4-6 · 備貨 → 出貨 → 驗收
- 業務的責任：**追進度 + 通知客戶**。出貨前一天打電話／LINE 客戶
- 物流：3 件以下走宅配（黑貓／新竹）；棧板貨走公司車或大型物流
- 驗收：**簽收 = 法律意義**。教客戶當場開箱驗，有問題當場拒簽

### Step 7-8 · 開票 + 收款
**「賣得出去」是初級業務，「收得回來」才是合格業務。**

- **票期管理**：30 / 60 / 90 天。手機行事曆設提醒
- **催收順序**：
  - 過期 1 天：LINE 提醒
  - 過期 7 天：電話
  - 過期 14 天：直接拜訪 + 主管知會
  - 過期 30 天：上報財會走法務流程
- **發票**：公司發票 + 三聯式。客戶要二聯就要事先講
- **業務鐵則**：沒收到錢的訂單只是「借錢給客戶讓他用」 — 業績是「貨出 + 錢進」，缺一不可

## 10. 三大表單作業

### 10.1 費用申請單
**業務帶客戶吃飯、加油、買樣品的報帳單據。**

- **可報**：客戶招待餐 / 通行費 / 樣品 / 印刷 / 雜支
- **不能報**：個人加油、私人聚餐
- **流程**：填表 → 主管核 → 財會核
- **關鍵**：發票統編 = 公司、抬頭 = 公司全名。**招待要寫被招待客戶名 + 案件名**，不寫財會打回

### 10.2 異折單
**客戶說「這批貨有問題，少算一點」的處理單。**

- 步驟 1：先確認問題（真的瑕疵？運送損壞？還是想凹？）拍照存證
- 步驟 2：跟主管報告（金額多少？品項？原因？）
- 步驟 3：填異折單（系統開單 → 主管簽核 → 財會處理 → 客戶帳上自動扣抵下次貨款）
- **絕對不要**：自己答應客戶「我幫你少算 5 千」 = 沒走系統 = 自己賠錢 + 主管罵

### 10.3 特案申請單
**當客戶條件超出標準時用。**

- **何時用**：
  1. 報價低於底價（毛利不足）
  2. 量大要求超優惠（如 1,000 套以上）
  3. 戰略性接單（虧本搶單，搶占案場）
- **必填**：
  1. 客戶背景（A 級／重點工程）
  2. 案件規模 + 後續潛力
  3. **不批的話誰會搶走**（競品威脅）
- **流程**：小額：主管核。中額：區經理。大額：業務總經理。**事前申請**，不能事後補單

## 11. 跨部門協作（業管／採購／財會）

### 11.1 業管（OPERATIONS · 你的後方司令部）
- **做什麼**：訂單 + 客戶資料 + 出貨
- **何時找**：下單前「庫存夠不夠？」／ 出貨後「貨到哪？」／ 客戶問「我的訂單編號？」
- **合作關鍵**：給他們完整資料 — 客戶名 / 統編 / 收件人手機 / 地址 / 特殊指示，**一次給全**

> 業管要每天下午跑出貨單，5 點後才丟急單會被白眼。**真有急事先打電話打招呼，再丟單。**

### 11.2 採購（PURCHASING · 控制庫存）
- **做什麼**：進貨 + 庫存 + 缺貨預警
- **何時找**：大單（超過 500 套以上）/ 客戶要急貨缺貨時
- **合作關鍵**：提早通知 + 信用 — 案場確定下單就先 LINE 採購預告
- **跟採購建立信用** = **承諾的單一定要中**。一次跳票兩次扣分，第三次他們自己會把預留庫存收回去

### 11.3 財會（FINANCE · 看每張單的真假）
- **做什麼**：應收 + 發票 + 傳票
- **何時找**：客戶逾期 / 發票要重開 / 費用核銷被退 / 月結有疑問
- **合作關鍵**：資料完整 + 守時 — 表單欄位填全（發票統編、客戶名、案件名）/ 25 號前交報帳

> 財會不是「擋你錢的人」，是「**確保公司不出包的人**」。態度好一點他們會主動提醒你哪裡會被查稅。

### 11.4 跨部門協作 · 黃金三原則
1. **不把他們當服務生** — 業務不是天王。業管／採購／財會的時間跟你一樣寶貴
2. **事前溝通 > 事後補救** — 大單先告知、缺貨先預警、報帳先問清楚 — 90% 內部摩擦都是「沒先講」造成
3. **幫他們解決問題** — 客戶資料填完整、單據填正確、催款主動跟、結案主動回 — 你幫他們省力，他們幫你開綠燈

> 業務在公司被討厭通常不是因為「績效不好」，是因為「**每次找人就是來丟事**」。
