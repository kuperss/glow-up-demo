## 2. 三安福祉（久任員工的長期保障辦法）

「三安福祉」是公司給予久任員工的長期保障辦法 — 從「人在職的安心」到「家庭的安身」，再到「退休後的安老」，建構展晟人的全生涯保障網。

### 2.1 安心
**在職期間的健康保障與工作環境支援**

照顧每一位用心耕耘的展晟人，讓你能無後顧之憂地專注工作。

### 2.2 安身
**重大事故支援與家庭保障**

當員工遭遇重大事故、家庭變故時，提供經濟與心理的支援機制。

### 2.3 安老
**退休生活照顧**

延續對久任員工的照顧到退休後，是公司對「以人為本」的具體承諾。

> **三安 = 安心（在職保障）+ 安身（重大事故支援）+ 安老（退休生活照顧）**
> 是展晟對「以人為本」的具體承諾 — **久任不只是年資累積，更是公司對你長期投入的具體回饋。**
> 詳細權益試算請洽各區人資暨法務部 HR。

## 4. 端午 / 中秋禮金

「正式員工享有所有福利待遇」 — 須**通過試用期評核**並核發時在職且實際從事工作者為限。

### 計算方式說明
- ★ **例**：試用 3 個月，第 2 個月就通過評核 = 已是正式員工，可享中秋禮金
- ★ **反之**：任職 3 個月但未通過評核 = 非正式員工，無法享有

## 8. 雙軌職涯路徑（CAREER PATH）

展晟採「管理職（Management）+ 專業職（Professional）」雙軌設計。
想帶人、想精專，兩條路都被公司認可，也都有對應的職等、薪酬與發展。

### 8.1 管理職軌道（Management Track）

管理職重視領導、組織、用人、規劃與管理能力，透過帶領團隊與整合資源創造組織價值。

| 職等 | 職稱 |
|---|---|
| M04 | 協理 |
| M05 | 經理 |
| M06 | 副理 |
| M07 | 課長 |
| **M08** | **組長（業務管理職入口）** |

### 8.2 專業職軌道（Professional Track）

專業職重視不同工作性質的專業知識與技能，用「無可取代的專業」創造組織價值。

| 職等 | 職稱 |
|---|---|
| P06 | 區副理 |
| P07 | 業務主任 |
| P08 | 資深專員 |
| P09 | 業務專員 |
| **P11** | **實習生、兼職人員（業務專業職入口）** |

### 8.3 業務職涯主軸

從 P11 實習生進入業務部，沿專業軌深耕：

```text
P11 實習生／兼職 → P09 業務專員 → P08 資深專員 → P07 業務主任 → P06 區副理
```

在資深專員（P08）或業務主任（P07）階段，可依能力與意願考慮跨入管理軌，往 M08 組長方向發展。

### 8.4 升等條件與時程

- 達一定職等以上之職務，須於現職**擔任屆滿規定年資**後，方得申請晉升。
- 職等晉升依公司年度規劃**定期核定**。
- 升等審查經**權責單位複核**，確認申請人符合職務說明資格與升等條件。
- 新聘入職時經核定之職務、職等，**不適用上述申請程序**。
- 申請時依公司規定之相關人事表單辦理。

### 8.5 核決流程

```text
權責主管提報 → 依規定程序審議 → 經權責單位核定
```

**資料來源**：展晟照明集團 雙軌職涯路徑表 v13（2024/01/25）
