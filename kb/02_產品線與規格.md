## 1. 七大應用場域（Application Domains）

舞光官方分類為**七大應用場域**，覆蓋從家庭到工業的所有照明需求：

### 1.1 居家空間（HOME）
- **照明風格**：主燈 + 間接光 + 重點光的層次照明
- **色溫建議**：2700K-3000K 暖光
- **常用產品**：
  - 客廳：雲朵吸頂燈
  - 臥室：Ai 智慧崁燈
  - 走道：索爾／奧丁崁燈
  - 氛圍：軟條燈鋁槽

### 1.2 商業空間（COMMERCIAL）
- **照明風格**：高演色重點打燈，凸顯商品質感
- **演色建議**：Ra ≧ 90、R9 > 50
- **常用產品**：
  - 服飾店：拉斐爾磁吸軌道
  - 餐廳：馬爾防眩崁燈
  - 咖啡廳：軌道投射燈
  - 展示櫃：軟條燈

### 1.3 辦公空間（OFFICE）
- **照明風格**：防眩、均光，避免螢幕反射
- **關鍵指標**：UGR < 19、4000K
- **常用產品**：
  - 主光源：輕鋼架平板燈
  - 會議室：馬爾防眩崁燈
  - 個人桌：護眼檯燈
  - 緊急照明（消防驗收）

### 1.4 學校照明（SCHOOL）
- **照明風格**：護眼低藍光，符合教育部 CNS 標準
- **關鍵指標**：UGR < 19、低藍光、Ra ≧ 80
- **常用產品**：
  - 教室：護眼平板燈
  - 走廊：吸頂燈／崁燈
  - 體育館：高天井燈
  - 宿舍：吸頂燈／壁燈

### 1.5 工廠照明（INDUSTRIAL）
- **照明風格**：高瓦數高流明，看清楚為主
- **關鍵指標**：高 lm/W、5000K-6500K
- **常用產品**：
  - 主光源：高天井燈
  - 食品廠：IP66 防潮
  - 停車場：防潮燈
  - 逃生：緊急照明

### 1.6 互聯數位（DIGITAL）
- **照明風格**：語音、APP、多情境一鍵切換
- **生態相容**：米家 / Google Home（不支援 Apple HomeKit）
- **常用產品**：
  - 智慧主燈：Ai 系列崁燈
  - 智慧吸頂：智能雲朵
  - 智控氛圍：軟條燈
  - 生態相容：米家 / Google

### 1.7 戶外空間（OUTDOOR）
- **照明風格**：耐候、防水、防雷擊
- **關鍵指標**：IP65 起跳、耐候塗裝
- **常用產品**：
  - 道路：高燈／路燈
  - 招牌：泛光燈（宙斯）
  - 庭園：草皮燈／照樹燈
  - 階梯：階梯燈／地底燈

## 4. 規格術語速查（SPEC 101 · 八個術語）

業務面對客戶最常被問的八個術語，這份是速查表。

### 4.1 流明（Lumen / lm）
**亮度單位 · lm**

客戶真正在乎的「夠不夠亮」就看這個。
- 10W LED 約 800-1000 lm，等同傳統 60W 白熾燈

### 4.2 色溫（Kelvin / K）
**光的顏色 · K**

| 色溫 | 感受 | 應用 |
|---|---|---|
| 2700K | 暖黃如夕陽 | 居家臥室、餐廳 |
| 3000K | 暖白如日落 | 居家客廳、咖啡廳 |
| 4000K | 自然如陰天 | 辦公室、商空主光 |
| 6500K | 正白如手術燈 | 工廠、戶外 |

### 4.3 演色性 Ra（CRI）
**還原物體真實顏色的能力 · 0-100**

| Ra | 應用 |
|---|---|
| Ra 80 | 居家、辦公夠用 |
| Ra ≧ 90 | 商空、餐廳必備（食物、衣服顏色更真實） |
| R9 > 50 | 紅色還原（服飾店標配） |

### 4.4 光束角（Beam Angle）
**光線打出去的擴散角度**

| 光束角 | 用途 |
|---|---|
| 15-30° | 重點打燈（投射燈、軌道燈） |
| 60-90° | 一般主光（崁燈） |
| 120-180° | 大面積均光（平板燈、吸頂燈） |

### 4.5 IP 防護等級
**防塵 + 防水**

| 等級 | 應用 |
|---|---|
| IP20 | 室內 |
| IP44 | 浴室、廁所 |
| IP65 | 戶外（淋雨） |
| IP66 | 戶外重型（強水柱、廠房） |

### 4.6 瓦數（Watt / W）
**耗電量 · 不直接代表亮度**

LED 時代要看「lm/W」（每瓦流明數），代表能效。
- 80 lm/W = 一般水準
- 100+ lm/W = 高效節能

### 4.7 壽命（Life Span）
**燈具光衰至 70% 的小時數**

舞光主流產品壽命 **15,000-25,000 小時**。
換算：每天用 8 小時，約 5-8 年。

### 4.8 驅動方式（Driver）
**控制電流穩定性，影響閃爍與壽命**

| 類型 | 特性 |
|---|---|
| PWM | 便宜易閃爍 |
| TRIAC | 居家用 |
| 0-10V | 商業空間用 |
| DALI | 高階商業 / 工程 |

舞光中高階款大多採用 **直流恆流驅動**，是賣點之一。
